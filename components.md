# DATA

## DATA LAYER

- Current number
- Former number

## DATA MODIFICATIONS

- Add digit to current number
- Clear all digits from current number
- Delete last digit from current number

- Add numbers
- Substract numbers
- Multiply numbers
- Divide numbers
- Give square root of a current number

## KEYBOARD

- Shows the received number
